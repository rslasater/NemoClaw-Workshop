# Project ATaS

Project ATaS is a two-hour workshop demonstrating Agentic AI using NVIDIA NemoClaw.

Students build an executive assistant that triages a realistic Outlook mailbox and prepares an executive brief for a congressional hearing.

The workshop contains:

- Fake Outlook web application
- Fake email REST API
- 235-email simulated Exchange mailbox
- Real Office attachments
- Instructor scoring guide
- Project ATaS scenario

See docs/ for all design documents.
